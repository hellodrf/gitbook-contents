# 欢迎来到VOLAMUS工作室！

![](.gitbook/assets/osprey-temp-banner.jpg)

VOLAMUS是一项纯原创“文字角色扮演”协同写作项目，背景设定为半架空世界的智能鸟类奇幻，主要面向青年成年人（YA）。我们创建自己的角色，然后通过“看帖回帖”的方式与其他成员的角色进行互动，共同叙述每个角色的故事。
